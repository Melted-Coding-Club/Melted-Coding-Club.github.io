Project 2.1: Pong

Objective

Create a simple two-player Pong game using Python and `pygame`.

Requirements:

  • (Note: this is the same as the previous project. Use objects for this one!)
  • The game should display a window with a Pong table layout.
  • Two paddles controlled by players (e.g., W/S for Player 1, Up/Down for Player 2).
  • A ball that moves and bounces off the paddles and screen edges.
  • Scoring system: A player scores a point when the ball passes the opponent's paddle.
  • First player to reach a set score wins.

Tools & Skills:

Python, `pygame`, Using objects (classes) for the first time

Logic Flow:

  1. Initialize `pygame` and create the game window.
  2. Create objects for the paddles and ball.
  3. Detect player input and move paddles accordingly.
  4. Move the ball and check for collisions.
  5. Update the score when a player scores.
  6. Restart the round or end the game when a player wins.
  7. All logic specific to the paddle or ball should be within it's class as methods.

Extra Features (Optional, not included in GitHub solution):

  • Add sound effects for ball bounces and scoring.
  • Increase ball speed over time to make the game harder.
  • Implement AI for a single-player mode.
  • Enhance visuals with animations and better graphics.
