Project 3: Snake

Objective

Create a classic Snake game using Python and `pygame`.

Requirements:

  • Display a game window with a grid-based play area.
  • The snake moves continuously in the chosen direction.
  • The player can control the snake using arrow keys (or WASD).
  • Eating food makes the snake grow.
  • The game ends if the snake collides with itself or the screen boundary.
  • Keep track of the player's score.

Tools & Skills:

Python, `pygame`, game loops, collision detection, keyboard input handling.

Logic Flow:

  1. Initialize `pygame` and create the game window.
  2. Draw the snake, food, and scoreboard.
  3. Detect player input and change the snake’s direction.
  4. Move the snake and check for collisions.
  5. Increase the snake’s length when it eats food.
  6. End the game if the snake hits the wall or itself.

Extra Features (Optional, not included in GitHub solution):

  • Add sound effects for eating food and collisions.
  • Increase speed over time to make the game more challenging.
  • Implement a high-score system.
  • Introduce obstacles or power-ups.
