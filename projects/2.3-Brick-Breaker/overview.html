Project 2.3: Brick Breaker

Objective

Build a Brick Breaker (Breakout-style) game using Python and `pygame`.

Requirements:

  • Display a paddle that the player can move left and right using arrow keys or A/D.
  • Launch a bouncing ball that breaks bricks on contact.
  • Lay out multiple rows of bricks at the top of the screen.
  • Detect collisions between the ball, bricks, paddle, and screen edges.
  • Remove bricks when hit by the ball.
  • End the game when the ball falls below the paddle.
  • Track and display the player's score.

Tools & Skills:

Python, `pygame`, object-oriented programming (classes), collision detection, game loops, user input handling.

Logic Flow:

  1. Initialize `pygame` and create the game window.
  2. Create a `Paddle` class to handle player movement and drawing.
  3. Create a `Ball` class that handles movement and bouncing.
  4. Create a `Brick` class, and generate a grid of bricks at the top.
  5. Move the ball each frame and check for collisions with walls, paddle, and bricks.
  6. Destroy bricks on contact and increase the score.
  7. End the game if the ball goes off the bottom of the screen.

Extra Features (Optional, not included in GitHub solution):

  • Make the ball bounce at different angles based on where it hits the paddle.
  • Add lives or a "Game Over" screen.
  • Include sound effects and music.
  • Power-ups like multi-ball, paddle size increase, or slow motion.
  • Add multiple levels with different brick layouts.
