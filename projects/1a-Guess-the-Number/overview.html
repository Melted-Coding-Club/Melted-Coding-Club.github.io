Project 1a: Guess the Number

Objective

Create a simple terminal number-guessing game using Python.

Requirements:

  • The program selects a random number within a specified range (e.g., 1 to 100).
  • The player inputs their guess.
  • The game provides feedback:
    • "Too high!" if the guess is too high.
    • "Too low!" if the guess is too low.
    • "Correct!" if the guess is right.
  • The game continues until the correct number is guessed.
  • Bonus: Track and display the number of attempts.

Tools & Skills:

Python, `random` module, loops, conditionals, user input handling.

Logic Loop:

  1. Import the `random` module.
  2. Generate a random number.
  3. Use a loop to take user input until they guess correctly.
  4. Print appropriate feedback.
  5. Display the number of attempts when the player wins.

Extra Features (Optional, not included in GitHub solution):

  • Allow the player to set their own number range.
  • Implement difficulty levels.
  • Add a leaderboard or high-score tracker.
