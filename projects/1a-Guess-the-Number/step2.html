Step 2 - Generating a Random Number

  • In this step, we generate a random number within a specified range using the random library and the randint function that it contains.
  • When you install Python, a standard library is automatically included, so you can immediately use libraries like random, math, os, and many others without needing to install them separately.
  • This means we do not need to install random, only import it
  • Libraries contain functions, to access them we call the name of the library and then the function seperated by periods (.).
