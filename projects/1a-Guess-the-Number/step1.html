Project 1a: Guess the Number

Step 1 - User Input

  • We can use the inbuilt input() function to request an input from the user within the terminal.
  • We can then print this value back to the user using the inbuilt print() function.
  • The input() function returns a string, so we must convert it to an integer using the int() function.
  • We can provide the input() function with a prompt to guide the user on what they should enter.
            
1
2 userInput = int(input("Enter a number between 0 and 100: "))
3 print(userInput)
