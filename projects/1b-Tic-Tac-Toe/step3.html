Step 3 - Adding the Game Loop

  • In this step, we implement a while loop to keep the game going until it is manually stopped (line 14). The loop will continuously prompt the user to pick a square until the game ends (and a break statement is used).
  • We handle player input as before, but now if a slot is already filled, we skip the turn by using the continue statement (line 18). This ensures the player doesn't lose their turn for selecting an invalid square.
  • After each move, we switch the current player. If it's player "x" (current_player == 0), we set it to player "o" (current_player == 1), and vice versa (lines 22-25).
  • We continue printing the board after each turn to keep the game up to date.

board = [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' ', ' ']

def print_board(board):
    print(f' {board[0]} | {board[1]} | {board[2]} ')
    print('---|---|---')
    print(f' {board[3]} | {board[4]} | {board[5]} ')
    print('---|---|---')
    print(f' {board[6]} | {board[7]} | {board[8]} ')

current_player = 0
player_symbols = ['x', 'o']
print_board(board)

while True:
    user_input = input("Choose your square: ")
    if board[int(user_input) - 1] == ' ':
        board[int(user_input) - 1] = player_symbols[current_player]
    else:
        print("Slot already filled!")
        continue

    if current_player == 0:
        current_player = 1
    elif current_player == 1:
        current_player = 0
    print_board(board)

The game will continue prompting players to choose their square. For instance, the board setup will look like this:

            
       |   |
    ---|---|---
       |   |
    ---|---|---
       |   |

     x |   |
    ---|---|---
       |   |
    ---|---|---
       |   |

     x |   |
