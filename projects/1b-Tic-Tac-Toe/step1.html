Project 1b: Tic Tac Toe

Step 1 - Setting Up the Board

  • We start by saving the game board as a list of strings. Initially, each position is empty, represented by a space.
  • Each space will later be filled with either "x" or "o" depending on the player's move.
  • Next, we create a function called print_board() to print the game board. This function will make it easier to display the board whenever needed during the game.
  • We use the print() function within print_board() to display the values at the appropriate positions on the board.
            
1 board = [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' ', ' ']
2
3 def print_board(board):
4     print(f' {board[0]} | {board[1]} | {board[2]} ')
5     print('---|---|---')
6     print(f' {board[3]} | {board[4]} | {board[5]} ')
7     print('---|---|---')
8     print(f' {board[6]} | {board[7]} | {board[8]} ')
9
10 print_board(board)

This will print the initial empty Tic-Tac-Toe board:

            
       |   |
    ---|---|---
       |   |
    ---|---|---
       |   |
