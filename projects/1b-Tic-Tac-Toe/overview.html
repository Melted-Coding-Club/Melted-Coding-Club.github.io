Project 1b: Tic-Tac-Toe

Objective

Create a simple terminal-based Tic-Tac-Toe game using Python.

Requirements:

  • The game should be played on a 3x3 grid.
  • Two players take turns entering their moves.
  • The game checks for a win or a draw after each move.
  • Winning conditions:
    • A player wins by placing three marks in a row, column, or diagonal.
  • If the grid is full and no one has won, the game ends in a draw.

Tools & Skills:

Python, loops, conditionals, lists, and functions.

Logic Flow:

  1. Initialize a 3x3 grid.
  2. Display the grid to the players.
  3. Players take turns selecting a position.
  4. Update and display the grid after each move.
  5. Check for a win or draw after each turn.
  6. Declare the winner or a draw when the game ends.

Extra Features (Optional, not included in GitHub solution):

  • Allow players to choose their symbols (e.g., X and O).
  • Implement AI for a single-player mode.
  • Enhance the interface with a graphical version using `pygame`.
