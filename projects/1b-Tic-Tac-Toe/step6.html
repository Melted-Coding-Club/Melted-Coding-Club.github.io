Project 1b: Tic-Tac-Toe

Step 6 - Using Functions to Encapsulate Logic

  • Now that we have our board and players set up, we need to implement input validation to make sure that players choose a valid, unoccupied square.
  • We’ll create a function called is_valid_input() that checks if the player's input corresponds to a valid board position (either a number from 1-9 or a coordinate like "a1", "b2", etc.).
  • If the input is valid, the game will proceed, and the current player's symbol ("x" or "o") will be placed in the corresponding position on the board.
  • If the square is already filled, the game will prompt the player to choose another square.
  • We also add a check_win() function to check if any player has won the game by checking for any winning combination on the board.
  • The game will continue until there’s a winner or all squares are filled (which would result in a draw).
  • After every turn, we switch to the next player by alternating between index 0 and 1 in the player_symbols list.
            
1 board = [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' ', ' ']  # Initialize the game board
2 player_symbols = ['x', 'o']  # Player symbols
3 current_player = 0  # Start with player 0 (x)
4
5 def print_board(board):
6     print(f' {board[0]} | {board[1]} | {board[2]} ')
7     print('---|---|---')
8     print(f' {board[3]} | {board[4]} | {board[5]} ')
9     print('---|---|---')
10     print(f' {board[6]} | {board[7]} | {board[8]} ')
11
12 def is_valid_input(user_input):
13     """Validates if input is a valid number or coordinate."""
14     return user_input in ['1', '2', '3', '4', '5', '6', '7', '8', '9', 'a1', 'a2', 'a3', 'b1', 'b2', 'b3', 'c1', 'c2', 'c3']
15
16 def convert_to_number(user_input):
17     """Converts coordinate input to board index."""
18     if user_input[0] == 'a':
19         return int(user_input[1]) - 1
20     elif user_input[0] == 'b':
21         return int(user_input[1]) + 2
22     elif user_input[0] == 'c':
23         return int(user_input[1]) + 5
24     return int(user_input) - 1
25
26 def check_win(board):
27     """Checks if a player has won the game."""
28     winning_combinations = [
29         [0, 1, 2], [3, 4, 5], [6, 7, 8],  # rows
30         [0, 3, 6], [1, 4, 7], [2, 5, 8],  # columns
31         [0, 4, 8], [2, 4, 6]  # diagonals
32     ]
33     for combo in winning_combinations:
34         if board[combo[0]] == board[combo[1]] == board[combo[2]] != ' ':
35             return True
36     return False
37
38 print_board(board)
39 while True:
40     user_input = input("Choose your square: ").lower()
41     if not is_valid_input(user_input):
42         print("Invalid input")
43         continue
44     board_position = convert_to_number(user_input)
45     if board[board_position] == ' ':
46         board[board_position] = player_symbols[current_player]
47     else:
48         print("Slot already filled")
49         continue
50
51     if check_win(board):
52         print_board(board)
53         print(f"Player {player_symbols[current_player]} won!")
54         break
55
56     current_player = 1 - current_player  # Switch players
57     print_board(board)

This will enable the players to take turns, making moves and checking if any player has won.
