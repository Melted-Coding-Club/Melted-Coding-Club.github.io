Project 6: Tetris

Objective

Create a simplified version of the classic game Tetris using Python and `pygame`.

Requirements:

  • Implement a 10x20 grid for gameplay.
  • Randomly generate tetrominoes (Tetris pieces) that fall from the top.
  • Allow players to move and rotate pieces using keyboard input.
  • Clear completed lines and update the score accordingly.
  • End the game when the pieces stack to the top of the grid.

Tools & Skills:

Python, `pygame`, game loops, collision detection, keyboard input handling, grid-based logic.

Logic Flow:

  1. Define the grid and tetromino shapes.
  2. Spawn a random tetromino at the top of the grid.
  3. Move the tetromino down at regular intervals.
  4. Detect collisions with the grid or other pieces.
  5. Allow user input for moving and rotating tetrominoes.
  6. Clear full lines and update the score.
  7. End the game if the stack reaches the top.
  8. At least 1 "Next Piece" should be visible.

Extra Features (Optional):

  • Add guides to make it more obvious where the tetromino will land,
  • Or introduce a ghost tetromino to show where the current piece will land.
  • Add different difficulty levels by increasing speed over time.
  • Implement a hold piece mechanic.
  • Include a leaderboard to track high scores. (Store info in a text file)
