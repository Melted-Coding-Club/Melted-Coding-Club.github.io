Project 3.3: Paper.io

Objective

Create a simplified version of the popular game Paper.io using Python and `pygame`.

Requirements:

  • Players control a square that moves in four directions.
  • Leaving a trail behind creates a new territory when a loop is completed.
  • Players lose if an opponent or themselves cross their trail before it is closed.
  • The game should display a score based on the player's territory size.
  • Implement basic enemy AI to add competition.

Tools & Skills:

Python, `pygame`, game loops, collision detection, keyboard input handling.

Logic Flow:

  1. Initialize `pygame` and create the game window.
  2. Draw the player, trails, and captured territory.
  3. Detect player input and move accordingly.
  4. Track the player's trail and check for completed loops.
  5. Detect collisions with enemies, self-trails, or boundaries.
  6. Update the score based on captured area.

Extra Features (Optional):

  • Add different enemy AI difficulty levels.
  • Implement power-ups for temporary speed boosts or invincibility.
  • Introduce different map layouts or obstacles.
  • Allow multiplayer functionality.
