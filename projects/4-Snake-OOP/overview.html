Project 4: Snake (OOP)

Objective

Refactor the classic Snake game using Object-Oriented Programming (OOP) principles in Python with `pygame`.

Requirements:

  • Use OOP to structure the game (e.g., classes for Snake, Food, and Game).
  • The code should support any number of snakes and/or apples at once.
  • The snake moves continuously in the chosen direction.
  • The player can control the snake using arrow keys (or WASD).
  • Eating food makes the snake grow.
  • The game ends if the snake collides with itself or the screen boundary.
  • Keep track of the player's score.
  • The snake should not be able to reverse direction and collide directly with itself.

Tools & Skills:

Python, `pygame`, Object-Oriented Programming (OOP), game loops, collision detection, keyboard input handling.

Logic Flow:

  1. Define classes for Snake, Food, and Game.
  2. Initialize `pygame` and create the game window.
  3. Draw the snake, food, and scoreboard.
  4. Detect player input and change the snake’s direction.
  5. Move the snake and check for collisions.
  6. Increase the snake’s length when it eats food.
  7. End the game if the snake hits the wall or itself.

Extra Features (Optional):

  • Add sound effects for eating food and collisions.
  • Increase speed over time to make the game more challenging.
  • Implement a high-score system.
  • Introduce obstacles or power-ups.
