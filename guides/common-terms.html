Glossary

Programming

  • Coding vs. Programming Coding is the act of writing instructions in a programming language, focusing on syntax and implementation. Programming is a broader process that involves planning, designing, testing, and maintaining software solutions. In short, coding is a part of programming. Read more
  • IDE An Integrated Development Environment (IDE) is a software application that provides comprehensive tools for software development, including a code editor, debugger, and build automation. Popular Python IDEs include PyCharm and VSCode. Read more
  • Runtime Runtime is the period during which a program is running. It refers to the environment in which a program is executed, including memory allocation and system resources. Read more
  • Compiler A compiler is a program that translates source code from a high-level programming language to a lower-level language (e.g., machine code). This conversion happens before execution. Read more
  • Interpreter An interpreter directly executes instructions written in a programming or scripting language without requiring them to be compiled into machine code. Python is an interpreted language. Read more
  • Syntax Syntax refers to the set of rules that define the structure of a programming language. It specifies how keywords, operators, and symbols can be combined to create valid statements. Read more
  • Algorithm An algorithm is a step-by-step procedure for solving a problem or performing a computation. It is a sequence of instructions that are followed to achieve a desired outcome. Read more

Coding

  • Variable A storage location paired with a name for holding data. In Python, variables are dynamically typed, meaning they can change type. Read more
  • Function A block of reusable code that performs a specific task. Functions can have parameters and return values. Read more
  • Loop A sequence of instructions that repeats until a condition is met. Python supports for and while loops. Read more
  • List An ordered collection of items, which can be of different types. Lists are mutable, meaning they can be modified after creation. Read more
  • Dictionary A collection of key-value pairs, similar to a real-world dictionary. Keys must be unique and immutable, while values can be of any type. Read more
  • Module A file containing Python code, which can include functions and variables. Modules allow you to organize and reuse code across multiple programs. Read more
  • Class A blueprint for creating objects. It defines attributes (data) and methods (functions). Classes enable Object-Oriented Programming in Python. Read more
  • Exception An error that occurs during program execution, disrupting the normal flow. Python uses try and except blocks for exception handling. Read more
  • Lambda An anonymous function defined using the lambda keyword. It can have multiple arguments but only one expression. Read more
  • Import A statement used to include external modules or libraries in a Python script. This allows for code reuse and organization. Read more
  • Decorator A function that wraps another function, enhancing or modifying its behavior. Decorators are widely used for logging, authentication, and more. Read more
  • Iterable An object capable of returning its elements one at a time, allowing it to be looped over. Examples include lists, tuples, and dictionaries. Read more
  • Recursion A function that calls itself in order to solve smaller instances of the same problem. Recursion is useful for problems like factorial and Fibonacci sequences. Read more
  • List Comprehension A concise way to create lists using a single line of code. It is generally faster and more readable than using traditional loops. Read more
