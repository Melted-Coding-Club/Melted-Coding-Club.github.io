Lesson 1

Data Types

There are many different ways to store Data. The different ways to do so are known as Data Types. These are the most common data types:
  • Integers, like 1 or 5432
  • Floats, like 3.12 or 870.23342
  • Strings, like "John" or "hello world
  • Booleans, like True or False
  • Arrays, like [1, 2, 3]
  • Dictionary, like {name: "John", age: 30}

Variables

Variables are used to store data values. They can be used to store values of any type.

Operators

Operators are used to perform operations on variables and values. The most common operators are:
  • Arithmetic operators, like +, -, *, /
  • Comparison operators, like ==, !=, >, <, >=, <=
  • Logical operators, like &&, ||, !
  • Assignment operators, like =, +=, -=, *=, /=

Statements

Statements are used to perform actions. The most common statements are:
  • Conditional statements, like if, elif, else
  • Loop statements, like for, while
  • Exception handling statements, like try, except, finally
