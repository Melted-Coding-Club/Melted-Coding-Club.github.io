Lesson 3 - Conditional Statements and Loops

Conditional Statements

Conditional statements are used to perform different actions based on different conditions. In Python, the most common conditional statements are if, elif, and else.

Here's an example:

            
1 age = 18
2 if age >= 18:
3     print("You are an adult.")
4 else:
5     print("You are a minor.")

This would output: You are an adult.

Nested and Multiple Conditions

You can check multiple conditions using elif or nest conditions inside each other. For example: 

            
1 score = 85
2 if score >= 90:
3     print("Grade: A")
4 elif score >= 80:
5     print("Grade: B")
6 else:
7     print("Grade: C")

This would output: Grade: B

Loops

Loops are used to repeat a block of code multiple times. In Python, the two main types of loops are:

  • for loops - Used for iterating over a sequence (e.g., a list, tuple, string, or range).
  • while loops - Repeats a block of code as long as a condition is true.

For Loops

Example of a for loop:

            
1 numbers = [1, 2, 3, 4, 5]
2 for num in numbers:
3     print(f"Number: {num}")

This would output:

Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

While Loops

Example of a while loop:

            
1 count = 0
2 while count < 5:
3     print(f"Count: {count}")
4     count += 1

This would output:

Count: 0
Count: 1
Count: 2
Count: 3
Count: 4
