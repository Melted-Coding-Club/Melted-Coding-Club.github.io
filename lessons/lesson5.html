Lesson 5 - Dictionaries

What is a Dictionary?

A dictionary is a collection of key-value pairs. Each key is unique and is used to access its corresponding value. Dictionaries are mutable, which means you can change them after creation.

In Python, dictionaries are defined using curly braces {}, with each key-value pair separated by a colon :.

Example:


1
2



student = {"name": "John", "age": 25, "is_student": True}
print(student)

This would output: {'name': 'John', 'age': 25, 'is_student': True}

Accessing and Modifying Dictionaries

You can access the value of a specific key using square brackets [] or the .get() method. 


1
2



name = student["name"]
print(name)

This would output: John

You can also modify values or add new key-value pairs:


1
2
3



student["age"] = 26
student["grade"] = "A"
print(student)

This would output: {'name': 'John', 'age': 26, 'is_student': True, 'grade': 'A'}
