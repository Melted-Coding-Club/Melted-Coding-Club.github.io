Lesson 2 - Functions

What is a Function?

A function is a block of organized, reusable code that performs a single, related action. Functions help break down code into smaller, manageable parts and avoid repetition.

In Python, a function is defined using the def keyword, followed by the function name and parentheses.

Built-in Functions

Python has several built-in functions that you can use straight away, including:

  • print() - Outputs data to the console
  • input() - Gets input from the user
  • int(), float(), str() - Type conversions
  • len() - Returns the length of an item
  • append() - Adds an element to a list

Defining and Calling Functions

You can create your own functions using def. Here's an example: 

            
1 def greet(name):
2     print(f"Hello, {name}!")
3
4 # Calling the function
5 greet("Alice")

This would output: Hello, Alice!

Function Parameters and Arguments

Functions can accept parameters, which are values you pass to the function. These are specified in the parentheses when defining the function.

For example:

            
1 def add(x, y):
2     return x + y
3
4 # Calling the function with arguments
5 result = add(3, 4)
6 print(result)

This would output: 7

Return Statements

The return statement is used to send a result back from a function. This allows you to store the result and use it later in your program.

For example:

            
1 def square(num):
2     return num * num
3
4 result = square(5)
5 print(result)

This would output: 25
