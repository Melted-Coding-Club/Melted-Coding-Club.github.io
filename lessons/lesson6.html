Lesson 6 - Introduction to Pygame

Introduction to Pygame

Pygame is an external library used to create games and multimedia applications in Python. It provides functionalities for graphics, sound, and input handling.

To start using Pygame, you need to import the module and initialize it:

                
                    1 import pygame
                    2 pygame.init()

Creating a Window in Pygame

In Pygame, you can create a window using the pygame.display.set_mode() function. 

                
                    1 screen = pygame.display.set_mode((800, 600))
                    2 pygame.display.set_caption("My First Game")

This creates a window with a width of 800 pixels and height of 600 pixels, titled "My First Game".

Event Handling in Pygame

To keep the window open and responsive, you need to handle events. The main game loop checks for events such as closing the window. 

                
                    1 while True:
                    2     events = pygame.event.get()
                    3     for event in events:
                    4         if event.type == pygame.QUIT:
                    5             pygame.quit()
                    6             exit()

This keeps the window open until the user closes it.
