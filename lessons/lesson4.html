Lesson 4 - Lists and Tuples

What are Lists?

A list is a collection of items in a particular order. Lists are mutable, meaning they can be changed after creation. In Python, lists are defined using square brackets [] and items are separated by commas.

Example:

            
1 fruits = ["apple", "banana", "cherry"]
2 print(fruits)

This would output: ['apple', 'banana', 'cherry']

Accessing List Elements

You can access elements in a list by their index. Remember, Python uses zero-based indexing, so the first item is at index 0. 

            
1 first_fruit = fruits[0]
2 print(first_fruit)

This would output: apple

Modifying Lists

Lists are mutable, so you can change their contents after creation. You can:

  • Change an element's value
  • Add elements using .append() or .insert()
  • Remove elements using .remove() or .pop()

Example:

            
1 fruits.append("orange")
2 print(fruits)

This would output: ['apple', 'banana', 'cherry', 'orange']

What are Tuples?

A tuple is similar to a list, but it is immutable, meaning it cannot be changed after creation. Tuples are defined using parentheses () instead of square brackets.

Example:

            
1 coordinates = (10, 20)
2 print(coordinates)

This would output: (10, 20)

Accessing Tuple Elements

Just like lists, you can access tuple elements using indexing: 

            
1 x = coordinates[0]
2 y = coordinates[1]
3 print(f"X: {x}, Y: {y}")

This would output: X: 10, Y: 20

Tuples vs Lists

Key differences:

  • Lists are mutable, tuples are immutable.
  • Lists use square brackets [], tuples use parentheses ().
  • Tuples are generally faster and use less memory than lists.
