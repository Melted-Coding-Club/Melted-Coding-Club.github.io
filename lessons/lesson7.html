Lesson 7 - Moving Objects in Pygame

Understanding Coordinates in Pygame

In Pygame, the screen is like a flipped Cartesian plane:

  • The top-left corner is (0, 0)
  • The x-axis increases to the right
  • The y-axis increases downward

Example:


1



position = (100, 200)

This represents a point 100 pixels to the right and 200 pixels down from the top-left corner.

Drawing Rectangles

In Pygame, rectangles can be drawn using the pygame.draw.rect() function. Rectangles are often used for game objects like players, enemies, or obstacles.

Example:


1
2
3



rect_color = (255, 0, 0)  # Red color
rect_pos = (100, 100, 50, 50)  # x, y, width, height
pygame.draw.rect(screen, rect_color, rect_pos)

Moving Rectangles

To move a rectangle, update its x or y position inside the game loop. This creates the illusion of movement. 


1

x = 100
y = 100

while True:
    x += 1  # Move right
    screen.fill((0, 0, 0))  # Clear screen
    pygame.draw.rect(screen, (255, 0, 0), (x, y, 50, 50))
    pygame.display.update()

This will move the rectangle to the right continuously.

Controlling Movement with Keyboard Input

You can control the movement of objects using keyboard input. In Pygame, this is done by checking events inside the game loop.

x, y = 200, 200
while True:
    events = pygame.event.get()
    for event in events:
        if event.type == pygame.KEYDOWN:
            if event.key == pygame.K_UP:
                y -= 5
            elif event.key == pygame.K_DOWN:
                y += 5
            elif event.key == pygame.K_LEFT:
                x -= 5
            elif event.key == pygame.K_RIGHT:
                x += 5
    pygame.draw.rect(screen, (255, 0, 0), (x, y, 50, 50))
    pygame.display.update()

This allows the rectangle to move in all four directions using the arrow keys.
